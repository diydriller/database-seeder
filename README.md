       An IntelliJ-based tool that generates fake data and inserts it into your database automatically.
        This plugin helps developers easily populate tables with realistic test data
<!-- Plugin description end -->

## Installation

- Using the IDE built-in plugin system:
  
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> > <kbd>Search for "database-seeder"</kbd> >
  <kbd>Install</kbd>

- Manually:

  Download the [latest release](https://github.com/diydriller/database-seeder/releases/latest) and install it manually using
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>
